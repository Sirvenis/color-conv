Understanding Color Codes

HEX (Hexadecimal)

HEX codes are 6-digit combinations of numbers (0-9) and letters (A-F) preceded by a hash (#). The first two digits represent red, the middle two green, and the last two blue. For example, #FF6600 means R:255, G:102, B:0 — a bright orange. HEX is the most common format in web development and CSS.

RGB (Red, Green, Blue)

RGB defines colors using three values from 0 to 255, representing the intensity of red, green, and blue light. This is an additive color model — combining all three at maximum (255,255,255) produces white, while (0,0,0) produces black. RGB is used in screens, cameras, and digital displays.

HSL (Hue, Saturation, Lightness)

HSL is often more intuitive for designers. Hue (0-360°) represents position on the color wheel (0=red, 120=green, 240=blue). Saturation (0-100%) controls vividness. Lightness (0-100%) controls brightness. Adjusting lightness while keeping hue and saturation constant creates natural color variations.

CMYK (Cyan, Magenta, Yellow, Key)

CMYK is a subtractive color model used in printing. Unlike RGB which adds light, CMYK subtracts brightness using ink. The "Key" component is black, added for depth and detail. Use CMYK when preparing designs for physical printing — the colors you see on screen (RGB) won't exactly match printed output (CMYK).

Web Accessibility and Color Contrast

The Web Content Accessibility Guidelines (WCAG) define minimum contrast ratios to ensure text is readable by people with visual impairments, including color blindness.

Why It Matters

Approximately 300 million people worldwide have color vision deficiency. Ensuring proper contrast makes your content accessible to everyone, improves SEO (Google favors accessible sites), and is required by law in many jurisdictions for government and public-facing websites.

Frequently Asked Questions

What is a HEX color code?
A HEX color code is a 6-digit combination of numbers and letters preceded by #, representing the amount of red, green, and blue in a color. For example, #FF0000 is pure red.
What is the difference between RGB and HEX?
RGB and HEX represent the same colors differently. RGB uses three numbers (0-255) for red, green, and blue. HEX is a compact way to write the same values using base-16 notation. RGB(255,0,0) equals #FF0000.
What is WCAG contrast ratio?
WCAG contrast ratio measures the difference in luminance between foreground and background colors. AA requires 4.5:1 for normal text, AAA requires 7:1.
How do I convert HEX to RGB?
Split the 6-digit HEX code into three pairs. Convert each pair from base-16 to decimal. For example, #FF8C00: FF=255, 8C=140, 00=0, so RGB is (255,140,0).
What is HSL color?
HSL stands for Hue (0-360 degrees on the color wheel), Saturation (0-100%, how vivid), and Lightness (0-100%, how bright). It's more intuitive for adjusting colors than RGB.
What is CMYK used for?
CMYK (Cyan, Magenta, Yellow, Key/Black) is used in print design. Unlike RGB which is additive (light), CMYK is subtractive (ink). Use CMYK for anything that will be physically printed.
Can I use any HEX code on websites?
Yes, all modern browsers support HEX, RGB, HSL, and HWB color formats. HEX is the most common format in web development and CSS.
How do I pick accessible colors?
Use our Contrast Checker tool to test foreground/background color pairs. Aim for WCAG AA compliance (4.5:1 ratio) for body text and AAA (7:1) for maximum accessibility.
